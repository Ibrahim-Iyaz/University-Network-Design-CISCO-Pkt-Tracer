# University-Network-Design-CISCO-Pkt-Tracer

Comprehensive Cisco Packet Tracer project simulating a university campus network. Includes IP addressing, VLAN configuration, printers, FTP server, and wireless access.

## 🚀 Project Objective
To design and simulate a functional university network including:
- VLAN segmentation
- FTP access
- Printer and wireless configuration
- Static/Dynamic Routing

## 🛠️ Technologies Used
- Cisco Packet Tracer
- Networking concepts (VLAN, IP Addressing, Routing)

## 🧪 How to Run the Project
1. Open the `.pkt` file in Cisco Packet Tracer.
2. Review configuration for each device.
3. Use simulation mode to test connectivity.
